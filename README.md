## Requirements

The following software is required for the complete workflow (from git clone to the running Docker Container).
The specified versions are the tested ones.

* JDK 17+
* Git 2.25.1+
* Apache Maven 3.6.3+
* Docker 20.10.12+
* Docker-compose 1.29.2+
